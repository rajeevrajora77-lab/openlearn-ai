# Infrastructure as Code (Terraform)

Terraform configurations for provisioning cloud infrastructure.

## Providers

- Google Cloud Platform (GKE)
- AWS (Optional)

## Resources

- Kubernetes clusters
- Cloud SQL (PostgreSQL)
- Redis (Memorystore)
- Object storage
- Load balancers
- DNS configuration

## Usage

```bash
# Initialize Terraform
terraform init

# Plan changes
terraform plan

# Apply changes
terraform apply

# Destroy infrastructure
terraform destroy
```
