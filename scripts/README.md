# Scripts

Utility scripts for OpenLearn AI development and operations.

## Database Scripts

- `init_db.sql` - Initial database setup
- `migrate.sh` - Run database migrations
- `seed.sh` - Seed database with test data

## Content Scripts

- `import_content.py` - Import educational content
- `generate_embeddings.py` - Generate embeddings for RAG

## Deployment Scripts

- `deploy.sh` - Deploy to production
- `rollback.sh` - Rollback deployment

## Monitoring Scripts

- `health_check.sh` - Check service health
- `backup.sh` - Backup databases

More automation scripts will be added as the project develops.
